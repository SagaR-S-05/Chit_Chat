## Installation

Follow these steps to set up and run the project:

### Prerequisites

- **Python 3.9+**
- **pip** (Python package installer)

### Clone the Repository

```bash
git clonehttps://github.com/18YashDB10/Chit_Chat.git
cd Chit_Chat
```

